User guide

This guide helps you to use this application. It contains information about how to manage the main functionality. If you have any doubt or problem while using it, please, contact us through the form placed in contact section.

The task list

The main view when you enter the application is the task list. In this view, you can find the tasks you added to the system. If the view has a deadline, it will be shown below the task. You can also see the percentage completed next to the title if the task was calified as complex.

If tasks are pending, you will find a couple of buttons next to them. The first one allows you to cancel the task, while the second one marks it as completed. A dialog to confirm the action will be shown in both cases.

The tasks are represented with different colours depending on their status and priority.

  • Red: High priority
  • Blue: Medium priority
  • Pink: Low priority
  • Green: Completed task.
  • Gray: Cancelled task

You can tap on any task to show its details in a new view. Besides, you can add a new task to the list by pressing the button on the right bottom.

Task details

The task details view shows every data of the task: title, priority, status and description. It will also shows the selected deadline if any exists and a progress interactive bar if it was marked as complex. You can increase or decrease this progress bar, and changes will be reflected automatically to the task.

If task is pending, you will see some options in the action bar too: edit task, complete task and cancel task. The first one will open a view to edit the information of the task. Both others have the same functionallity of the previous ones explained in the task list view.

Creating and updating a task

Whether you choosed to create a new task or editing an existing one, the same view will be shown. You will see a form to be filled. If you check the task as "has deadline", the finish button will change to next.

Pressing next button, you will see a date picker to select the deadline date. From this view, you can return to the previous one, cancel the action or complete by pressing the finish button.
